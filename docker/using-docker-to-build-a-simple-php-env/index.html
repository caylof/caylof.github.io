使用 docker 搭建简单 PHP 环境

项目目录结构如下：

目录 conf.d 中是 nginx 的 vhosts 配置文件，目录 www 为 php 存放文件。

php-fpm 部署

docker run 命令参数说明：

  • --name 将容器命名为 my-php-fpm。
  • -d 守护进程。
  • -v 将项目中的 /workspace/test-app/www 目录映射到容器中 /var/www/html 目录。
  • php:7.4.3-fpm 是创建容器所使用的镜像（REPOSITORY:TAG）。

nginx 部署

docker run 命令参数说明：

  • --name 将容器命名为 my-nginx。
  • -p 将宿主机的8080端口映射到容器的80端口。
  • -d 守护进程。
  • -v 映射目录。
  • --link 将要创建的容器与另一个容器（my-php-fpm）相连，my-php-fpm:php是将php作为my-php-fpm的一个别名，若在该容器要使用 my-php-fpm 容器，则可用别名php替代。（比如在 nginx 的配置文件当中 fastcgi_pass php:9000）
  • nginx 是创建容器所使用的镜像。
