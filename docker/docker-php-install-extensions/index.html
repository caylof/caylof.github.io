docker php 安装扩展

一般来说，官方的 php docker 镜像中只有一些基础的扩展，不过 php 镜像还提供了一些助手脚本（helper scripts）来帮助我们更方便地安装更多的扩展。

助手脚本类型于 docker-php-* 这样的格式，经常用到的：

  • docker-php-ext-configure ，类似于源码安装时的编译参数配置 ./configure --参数。
  • docker-php-ext-install ，类似于源码安装时的 make && make install。
  • docker-php-ext-enable ，启用扩展。
  • docker-php-source extract 解压 php 的源码包。
  • docker-php-source delete 将使用 docker-php-source extract 解压的目录删除。

通常安装扩展有三种方式：

  1. 安装 PHP 核心扩展
  2. 源码编译安装
  3. pecl 安装扩展（本文不展开介绍）

由于本篇文章使用 php:<version>-alpine 镜像，着重介绍前两种安装方式，常用到两个脚本：

  • docker-php-ext-configure
  • docker-php-ext-install

安装 PHP 核心扩展

最简单的安装

PHP 的核心扩展安装文件都在 PHP 源码包中，使用 docker-php-ext-install 来进行安装，例如安装 pdo_mysql 扩展。
